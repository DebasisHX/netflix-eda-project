# Netflix Movies & TV Shows EDA Project

## Project Description
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset to identify trends in content type, growth, genres, countries, and viewing patterns using Python.

## Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  

## Key Insights
- Movies significantly outnumber TV Shows on Netflix  
- Content growth increased rapidly after 2015  
- International Movies, Dramas, and Comedies dominate the catalog  
- United States, India, and United Kingdom produce most content  
- Most movies are 80–120 minutes long  
- Most TV shows have 1–2 seasons
